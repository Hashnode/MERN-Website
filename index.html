MERN

Easily build production ready universal apps.

Two awesome ways to get started :

MERN Starter

v2.0

A powerful boilerplate project that gives you a solid head start to building universal React apps.

  
    npm install -g mern-cli
    mern init your_new_app
    cd your_new_app
    npm install
    npm start

MERN cli

v3.0

A command line utility that helps you work with your MERN based projects easily by providing powerful code generation and scaffolding options.

  
    npm install -g mern-cli
    mern init myApp
    cd myApp
    npm install
    npm start
