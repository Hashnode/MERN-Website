What's MERN?

MEARN is a scaffolding tool which makes it easy to build isomorphic apps using solid technologies like Mongo, Express, React and NodeJS. It minimizes the setup time and gets you up to speed using proven technologies.

React is a JavaScript library for creating user interfaces by Facebook and Instagram. Many people choose to think of React as the V in MVC. React solves one problem: building large applications with data that changes over time.

Express.js is a Node.js web application server framework, designed for building single-page, multi-page, and hybrid web applications. It is the de facto standard server framework for node.js.
